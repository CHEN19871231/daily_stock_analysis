## ⚙️ 个性化调整

### 1. 修改股票列表
编辑 `.env` 文件，修改 `STOCK_LIST`：
```bash
# 示例：添加你关注的短线股票
STOCK_LIST=000001,600036,300059,002415,300750
```

### 2. 调整交易策略
编辑 `.env` 文件，修改 `AGENT_SKILLS`：
```bash
# 短线交易推荐策略
AGENT_SKILLS=bull_trend,ma_golden_cross,volume_breakout,shrink_pullback,bottom_volume

# 或启用所有策略
AGENT_SKILLS=all
```

### 3. 调整分析时间
对于短线交易，可能需要更频繁的分析：
```bash
# 每天多次分析（如开盘、午盘、收盘）
SCHEDULE_TIME=09:30,11:30,15:00
```

## 🎯 运行方式选择

### 方式一：GitHub Actions（推荐）
1. 将项目推送到你的GitHub仓库
2. 在GitHub仓库Settings → Secrets中确认配置
3. 启用Actions工作流
4. 系统会自动在交易日18:00运行

### 方式二：本地定时运行
```bash
# 启动定时任务
python3 main.py --schedule

# 或使用系统crontab
crontab -e
# 添加：0 18 * * 1-5 cd /path/to/daily_stock_analysis && python3 main.py
```

### 方式三：Docker运行
```bash
# 构建镜像
docker build -t stock-analysis .

# 运行容器
docker run -d --name stock-analysis stock-analysis
```

## ⚠️ 注意事项

1. **API用量**：DeepSeek有免费额度，注意监控使用量
2. **交易日**：默认只在交易日运行，可强制运行：`python3 main.py --force-run`
3. **数据延迟**：免费数据源可能有15分钟延迟
4. **投资建议**：仅供参考，不构成投资建议

## 🆘 故障排除

### 常见问题：
1. **依赖安装失败**：尝试使用Python虚拟环境
2. **API调用失败**：检查API Key是否正确，网络是否通畅
3. **飞书收不到消息**：检查Webhook地址，飞书机器人是否启用
4. **数据获取失败**：尝试更换数据源（akshare/yfinance）
