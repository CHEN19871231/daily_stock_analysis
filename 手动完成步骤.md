# 🚀 手动完成GitHub部署步骤

由于Git推送需要认证，以下是需要你手动完成的步骤：

## 步骤1：获取GitHub Personal Access Token

1. **访问**：https://github.com/settings/tokens
2. **点击**："Generate new token" → "Generate new token (classic)"
3. **设置**：
   - Note: `daily_stock_analysis_push`
   - Expiration: 90 days (推荐)
   - Select scopes: 勾选 `repo` (全选)
4. **生成**：点击 "Generate token"
5. **复制**：保存好生成的token（以`ghp_`开头）

## 步骤2：使用token推送代码

在终端中执行：

```bash
# 进入项目目录
cd /root/.openclaw/workspace/daily_stock_analysis

# 使用token设置远程仓库
git remote set-url origin https://CHEN19871231:你的token@github.com/CHEN19871231/daily_stock_analysis.git

# 推送代码
git push origin main
```

## 步骤3：配置GitHub Secrets（网页操作）

1. **访问仓库**：https://github.com/CHEN19871231/daily_stock_analysis
2. **进入设置**：Settings → Secrets and variables → Actions
3. **添加Secrets**（点击 "New repository secret"）：

## 步骤4：启用GitHub Actions

1. **访问Actions**：https://github.com/CHEN19871231/daily_stock_analysis/actions
2. **启用工作流**：点击 "I understand my workflows, go ahead and enable them"
3. **手动运行测试**：
   - 点击 "每日股票分析"
   - 点击 "Run workflow"
   - 选择模式：`full`
   - 点击 "Run workflow"

## 步骤5：验证部署

1. **等待运行完成**（约2-5分钟）
2. **查看运行日志**：确认无错误
3. **检查飞书**：是否收到分析报告
4. **确认自动化**：明天18:00会自动运行（交易日）

2. **Secrets配置失败**：确认在正确的仓库中配置
