# Daily Stock Analysis 配置说明

## 项目概述
这是一个基于AI大模型的A股/港股/美股自选股智能分析系统，每日自动分析并推送「决策仪表盘」。

## 已完成的配置

### 1. 基础配置
- **股票列表**：已设置为常见A股龙头股（600519贵州茅台、300750宁德时代、002594比亚迪、600036招商银行、601318中国平安）
- **定时任务**：每天18:00自动运行（交易日）
- **交易日检查**：已启用（非交易日不运行）

### 2. AI模型配置（需要你补充）
**推荐方案：DeepSeek API（性价比高，有免费额度）**

需要你完成：
1. 访问 https://platform.deepseek.com/api_keys 注册并获取API Key
2. 在 `.env` 文件中替换 `OPENAI_API_KEY=你的DeepSeek_API_Key_在这里`

### 3. 通知渠道配置（需要你补充）
**推荐方案：飞书机器人**

需要你完成：
1. 在飞书中创建机器人：
   - 方式一：在飞书群中添加自定义机器人
   - 方式二：在飞书开放平台创建应用并添加机器人能力
2. 获取Webhook地址
3. 在 `.env` 文件中替换 `FEISHU_WEBHOOK_URL=你的飞书Webhook地址在这里`

### 4. 高级功能配置
- **Agent对话模式**：已启用（支持11种交易策略）
- **默认策略**：多头趋势、均线金叉、放量突破、缩量回踩
- **新闻时效**：3天内（避免过时信息）
- **乖离率阈值**：5%（超过提示风险）

## 下一步操作

### 选项一：GitHub Actions部署（推荐）
1. 将你的API Key和Webhook地址添加到GitHub Secrets
2. 启用GitHub Actions工作流
3. 系统会自动在每天18:00运行分析

### 选项二：本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行测试
python test_env.py

# 手动运行分析
python main.py

# 启动Web界面（包含Agent对话功能）
python main.py --webui
```

### 选项三：Docker运行
```bash
# 构建镜像
docker build -t daily-stock-analysis .

# 运行容器
docker run -d --name stock-analysis daily-stock-analysis
```

## 自定义配置建议

### 股票列表调整
编辑 `.env` 文件中的 `STOCK_LIST`，添加你关注的股票代码：
- A股格式：600519,300750,002594
- 港股格式：hk00700,hk00981
- 美股格式：AAPL,TSLA,MSFT

### 交易策略调整
编辑 `.env` 文件中的 `AGENT_SKILLS`，选择适合的策略：
- `bull_trend` - 多头趋势
- `ma_golden_cross` - 均线金叉  
- `volume_breakout` - 放量突破
- `shrink_pullback` - 缩量回踩
- `chan_theory` - 缠论分析
- `wave_theory` - 波浪理论

### 通知频率调整
- 单股推送模式：设置 `SINGLE_STOCK_NOTIFY=true`
- 报告类型：`REPORT_TYPE=simple`（精简）或 `full`（完整）
- 仅推送摘要：设置 `REPORT_SUMMARY_ONLY=true`

## 注意事项
1. **API Key安全**：不要将 `.env` 文件提交到GitHub
2. **免费额度**：DeepSeek有免费额度，注意使用量
3. **交易日**：系统默认只在交易日运行，可强制运行：`python main.py --force-run`
4. **投资建议**：系统分析仅供参考，不构成投资建议
